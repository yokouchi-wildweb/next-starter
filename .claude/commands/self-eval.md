# ベーステンプレート自己評価

以下のファイルを読み込んで評価を開始してください：

1. @docs/self-evaluation/評価設定.json - AI向け評価設定（項目・チェックポイント・参照先）
2. @docs/self-evaluation/!評価基準.md - 評価基準と運用ガイドライン

## 実行手順

1. まず評価設定.jsonを読み込み、評価項目とチェックポイントを把握する
2. ユーザーに評価モードを確認する：
   - **俯瞰評価**: 全9項目を浅く確認し仮スコアを付与
   - **項目別評価**: 1項目ずつ詳細に評価（新規セッション推奨）
3. 過去の評価結果がある場合は `docs/self-evaluation/` 内の最新の `YYYY-MM-DD.md` を参照
4. 評価完了後、結果を `docs/self-evaluation/YYYY-MM-DD.md` に保存

## 評価スケール

- A: 完成（本番利用可能）
- B: 実用可（軽微な課題あり）
- C: 開発中（一部未実装）
- D: 未成熟（追加作業必要）
- E: 未着手

## 注意事項

- コードの静的レビューのみで評価する（動作確認は任意）
- 各項目の参照先ファイルを実際に読んで評価する
- 項目別評価の場合、各項目完了後にレビューチェックリストを実施する
